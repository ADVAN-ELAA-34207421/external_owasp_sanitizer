OWASP Java HTML Sanitizer Change Log

  1. Added convenient pre-packaged policies in Sanitizers. Fixed bug in how warnings are reported via the badHtml Handler.
  2. Better handling of supplementary codepoints to avoid UTF-16/UCS-2 confusion in browsers.
  3. Added new HTML5 URL attributes to list used to safeguard URL attributes in HtmlPolicyBuilder.
  4. Changed HtmlSanitizer.sanitize to allow null as a valid value for the HTML snippet.
